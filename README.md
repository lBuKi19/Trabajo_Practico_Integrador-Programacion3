# Trabajo_Practico_Integrador-Programacion3

LINK A VIDEO EXPLICATIVO: 

Proyecto: Food Store
Materia: Programación III
Tecnologías principales: TypeScript, Vite, HTML, CSS, Java, JPA, Hibernate, H2, Gradle
Modalidad: Frontend Web + Backend de consola con persistencia JPA/Hibernate.

El proyecto Food Store permitió integrar dos enfoques complementarios de desarrollo. Por un lado, el frontend web permitió construir una experiencia completa de usuario, incluyendo autenticación, catálogo, carrito, checkout, historial de pedidos y administración. Por otro lado, el backend con Java, JPA, Hibernate y H2 permitió aplicar conceptos de persistencia real, relaciones entre entidades, repositorios genéricos, consultas JPQL, baja lógica y transacciones atómicas.

INSTRUCCIONES PARA CORRER FRONTEND

Requisitos previos: instalar node.js y pnpm
- Clonar repositorio: git clone https://github.com/lBuKi19/Trabajo_Practico_Integrador-Programacion3.git
- Acceder al directorio via:  cd nombre-proyecto
- Instalar dependencias: pnpm install
- Ejecutar servidor: pnpm dev
